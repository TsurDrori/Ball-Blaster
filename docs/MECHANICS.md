# Ball Blast — מכניקות המשחק המלאות

## תיאור כללי

Ball Blast הוא משחק ארקייד היפר-קז'ואל שפותח על ידי Voodoo. השחקן שולט בתותח בתחתית המסך שיורה אוטומטית זרם כדורים כלפי מעלה. כדורים ממוספרים (אויבים) יורדים מלמעלה. כל פגיעה מקטינה את המספר ב-1; כשמגיע ל-0 הכדור מתפוצץ. אויבים שמגיעים לתחתית מסיימים את המשחק.

---

## מכניקת התותח (Cannon)

| תכונה | פרטים |
|---|---|
| מיקום | קבוע בתחתית המסך, נע שמאל/ימין |
| תנועה | עקוב אחר עכבר (Mac desktop) |
| כיוון ירי | אוטומטי — יורה ישר למעלה |
| קצב ירי | ירי רציף אוטומטי |
| שדרוגים | קצב ירי, עוצמת כדור, ירי מרובה, גודל כדור |

**מודל תנועה:** `position.x = lerp(current_x, mouse_x, speed * delta)`, חסום בין `[cannon_width/2, screen_width - cannon_width/2]`.

---

## מכניקת הקליע (Bullet)

- זז ישר למעלה במהירות קבועה (אין כבידה על קליעים)
- רדיוס קבוע (גדל עם שדרוגים)
- נהרס בנגיעה בכדור אויב
- מספר קליעים בו-זמנית בגובה גבוה
- עם שדרוג multi-shot: 2–5 כדורים בפיצוץ אחד בזווית פן

---

## מכניקת כדורי האויב (Enemy Balls)

| תכונה | פרטים |
|---|---|
| צורה | עיגול (ספירה עם הצללה תלת-ממדית) |
| מספר התחלתי | עולה עם הגל — מתחיל נמוך (5–20) ומגיע לאלפים |
| מהירות ירידה | מהירות קבועה כלפי מטה, עולה עם הקושי |
| כוח עמידות (HP) | שווה למספר המוצג |
| נזק לפגיעה | 1 לפגיעה (שדרוגים יכולים להגדיל) |
| התנגשות עם קירות | קפיצה אלסטית — זווית פגיעה = זווית ניתוז |
| פיצול | בגרסאות מסוימות כדורים גדולים מתפצלים לשניים קטנים |
| נפילת מטבעות | בהרס, מטבעות מתפזרים כלפי מטה |

**צבע קידוד קושי:**
- ירוק: HP נמוך
- צהוב: בינוני
- כתום: גבוה
- אדום: מאוד גבוה
- סגול: קיצוני

---

## מערכת המטבעות (Coin System)

- אויבים מפילים מטבעות במותם
- ערך מטבע = פרופורציה ל-HP ההתחלתי
- מטבעות נאספים אוטומטית כשנוגעים בזון האיסוף מעל התותח
- מטבעות הם מטבע השדרוג היחיד

---

## מערכת השדרוגים (Upgrade System)

| שדרוג | אפקט | רמות אופייניות |
|---|---|---|
| קצב ירי (Fire Rate) | מפחית זמן בין יריות | 10–20 |
| עוצמת כדור (Ball Power) | מגדיל נזק לפגיעה | 10–20 |
| ירי מרובה (Multi-shot) | מוסיף כדורים לכל מחזור ירי | 3–5 |
| גודל כדור (Ball Size) | מגדיל רדיוס קליע | 5–10 |

**עלות שדרוג:** `base_cost * level^1.8` (עלייה אקספוננציאלית)

שדרוגים ניתנים לרכישה בכל זמן במהלך המשחק — ברגע שיש מספיק מטבעות.

---

## עליית קושי (Difficulty Scaling)

```
HP כדור אויב בגל N:  base_hp * N^1.5
מהירות ירידה בגל N: base_speed + N * speed_increment
מטבעות להריגה:       פרופורציה ל-HP (שומר על איזון כלכלי)
כדורים בו-זמנית:     גדל עם הגל
```

בגיים לייט: מספרים על הכדורים מגיעים למיליונים.

---

## תנאי סיום המשחק

כל כדור אויב שמגיע לגבול התחתון של אזור המשחק → **Game Over מיידי** (אין חיים).

---

## מבנה הסשן

1. **לפני הגל:** צפייה בשדרוגים, מספר גל
2. **משחק פעיל:** כדורים יורדים, תותח יורה, מטבעות נאספים, שדרוגים זמינים בכל עת
3. **אחרי מוות:** ניקוד (גל הגבוה ביותר), אפשרות לנסות שוב

## ישויות המשחק ותכונותיהן

### Cannon (תותח)
```
position.x:     float  - מיקום אופקי נוכחי
fire_rate:       float  - שניות בין יריות (0.15s → 0.05s עם שדרוגים)
damage:          int    - נזק לפגיעה (בסיס: 1)
bullet_count:    int    - כדורים לכל מחזור ירי (1–5)
spread_angle:    float  - זווית פן לירי מרובה (מעלות)
bullet_radius:   float  - רדיוס קליע
bullet_speed:    float  - מהירות כלפי מעלה (פיקסלים/שנ')
coins:           int    - יתרת מטבעות
upgrade_levels:  dict   - רמת כל שדרוג
```

### Bullet (קליע)
```
position:   Vector2 - מיקום נוכחי
velocity:   Vector2 - מהירות כלפי מעלה (+ זווית לירי מרובה)
radius:     float   - רדיוס התנגשות
damage:     int     - ירש מהתותח בזמן היצירה
lifetime:   float   - זמן חיים מקסימלי (למניעת דליפות)
```

### EnemyBall (כדור אויב)
```
position:     Vector2 - מיקום נוכחי
velocity:     Vector2 - כלפי מטה + קפיצות אופקיות
radius:       float   - גודל (גדל עם max_hp)
max_hp:       int     - HP התחלתי
current_hp:   int     - HP נוכחי
coin_value:   int     - מטבעות בהריגה
color:        Color   - צבע לפי טווח HP
```

### Coin (מטבע)
```
position:  Vector2 - מיקום נוכחי
velocity:  Vector2 - מהירות פיזור ראשונית
value:     int     - כמות מטבעות
gravity:   float   - תאוצה כלפי מטה
```

### WaveManager (מנהל גלים)
```
current_wave:       int   - מספר גל נוכחי
balls_this_wave:    int   - כדורי אויב בגל
hp_multiplier:      float - מכפיל HP לגל
speed_multiplier:   float - מכפיל מהירות
spawn_interval:     float - זמן בין הופעות
```

## אלמנטי UI/UX

### HUD תוך-משחקי
| אלמנט | מיקום | תוכן |
|---|---|---|
| מונה מטבעות | שמאל-עליון | אייקון מטבע + מספר |
| מספר גל | מרכז-עליון | "Wave 12" |
| לוח שדרוגים | רצועה תחתית | 2–4 כפתורי שדרוג עם עלות |

### כפתורי שדרוג
- שם שדרוג + רמה נוכחית
- עלות במטבעות
- ירוק = אפשרי, אפור = לא מספיק מטבעות
- אנימציית קנה-מידה בקנייה

### מסך Game Over
- כיסוי מוחשך
- כותרת "GAME OVER"
- גל שהגיע (ניקוד)
- שיא אישי
- כפתורי "נסה שוב" / "בית"
