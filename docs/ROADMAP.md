# Ball Blast — מפת דרכים לפיתוח

## סקירה כללית

פיתוח המשחק מחולק ל-5 מיילסטונים. כל מיילסטון מסתיים עם גרסה פועלת של המשחק.

---

## מיילסטון 1 — Prototype (שבוע 1)

**מטרה:** משחק בסיסי שניתן לשחק

### משימות:
- [ ] התקן Godot 4 ויצור פרויקט בסיסי
- [ ] יצור Main.tscn עם גבולות מסך וצבע רקע
- [ ] הטמע Cannon.tscn — תנועה שמאל/ימין עם העכבר
- [ ] הטמע Bullet.tscn — ירי אוטומטי כלפי מעלה
- [ ] הטמע EnemyBall.tscn — יורד, קפיצה מקירות
- [ ] מערכת HP + הפחתה עם פגיעות
- [ ] תנאי Game Over (כדור מגיע לתחתית)
- [ ] תנאי Destroy (HP = 0)

**תוצאה:** ניתן לירות ולהרוס כדורים, משחק נגמר אם כדור מגיע לתחתית

## תוספות אופציונליות (Post-Release)

- **Leaderboard מקומי** — שמירת 10 תוצאות הגבוהות ביותר

- **אפקט split balls** — כדורים גדולים מתפצלים ל-2

- **Daily Challenge** — גל ספציפי לאתגר יומי
- **Analytics** — מעקב אחר גל ממוצע לשיפור איזון קושי

### סשן 2 — מה נבנה (17/3/2026)

טכנולוגיה: HTML5 + JavaScript Canvas (ללא התקנות)

קבצים שנוצרו ב- `game/`:
- `index.html` — פתח בדפדפן לשחק
- `gameState.js` — מצב גלובלי + מערכת שדרוגים
- `cannon.js` — תותח עם smooth follow + ירי אוטומטי + muzzle flash
- `bullet.js` — קליעים עם glow effect
- `enemy.js` — כדורי אויב עם 3D shading + קידוד צבע לפי HP
- `coin.js` — מטבעות עם כבידה + איסוף אוטומטי
- `waveManager.js` — ניהול גלים עם עליית קושי
- `ui.js` — HUD + מסך Game Over + הכרזת גל
- `game.js` — לולאת משחק ראשית + particles + collisions
- `style.css` — עיצוב כהה עם כפתורי שדרוג

**הרצה:** לחץ פעמיים על `game/index.html`
